# Site internet pour CamCakes.

https://camcakes.ca/

Design: [HTML5 UP](http://html5up.net)

## Installation

1. Rouler `npm install`
2. Exécuter `node server.mjs`

## Récupérer un token Instagram
1. Aller sur le site `https://developers.facebook.com/`
2. Aller dans `My Apps` en haut à droite et se connecter
3. Choisir `Entreprise`
4. Dans l'onglet `Instagram`, choisir `API setup with Instagram login`
5. Agrandir la section 1, faire `Generate Token` et le copier au bon endroit.
